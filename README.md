## ğŸ“– What is Caesar Cipher?
The Caesar Cipher is one of the oldest known encryption techniques.  
It works by **shifting letters** of the alphabet by a fixed number (key).  
- Example with shift = 3:  
  - Plaintext: `HELLO`  
  - Ciphertext: `KHOOR`
